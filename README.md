# Media Asset Manager

> Version: 0.1.0 — Pre-Development
> Status: SDLC Stage 1 — Project Definition

## Overview

Media Asset Manager is a cross-platform desktop application that allows content creators to index, tag, and search media assets stored on removable drives or network shares — including when those drives are offline.

## Platforms

- macOS
- Windows
- Linux

## Tech Stack (Proposed)

- **Frontend:** React (via Tauri)
- **Backend/Shell:** Tauri (Rust)
- **Database:** SQLite (via local app data directory)
- **Media Processing:** TBD (Stage 3)
