# User Stories — Sprint 10

**Version:** 1.0  
**Date:** 2026-05-22  
**Sprint:** 10 — Tags, Asset Metadata, and Transcription

---

## Tags

**US-201**  
As a user, I want to create named tags so I can categorise my media assets.

**US-202**  
As a user, I want to apply tags to an asset from the detail pane so I can organise my library without leaving the current view.

**US-203**  
As a user, I want to remove a tag from an asset so I can correct mistakes without deleting the tag itself.

**US-204**  
As a user, I want to filter the library by one or more tags so I can quickly find related assets.

---

## Asset Metadata

**US-211**  
As a user, I want to write a description for an asset so I can record context about what the footage contains.

**US-212**  
As a user, I want to record a location for an asset so I can remember where footage was captured.

**US-213**  
As a user, I want description and location to appear in search results so I can find assets by what they contain, not just their filename.

---

## Transcription

**US-221**  
As a user, I want to download a Whisper model from within the app so I don't have to find and install it manually.

**US-222**  
As a user, I want to see which models are installed and delete ones I no longer need so I can manage disk space.

**US-223**  
As a user, I want a Generate Transcript button on video and audio assets so I can request transcription when I need it without it happening automatically.

**US-224**  
As a user, I want to choose the model, language, and provide an optional prompt before transcribing so I can optimise accuracy for my content.

**US-225**  
As a user, I want to see an estimated transcription time before I start so I can decide whether to proceed now or later.

**US-226**  
As a user, I want transcription to run in the background so I can keep working while it processes.

**US-227**  
As a user, I want to cancel a transcription in progress so I can stop it if I change my mind.

**US-228**  
As a user, I want to see the transcript as timestamped segments so I can quickly scan the content.

**US-229**  
As a user, I want to click a transcript segment to jump to that point in the video so I can review specific moments quickly.

**US-230**  
As a user, I want the search bar to find assets by transcript content so I can locate footage by what was said.

**US-231**  
As a user, I want to re-transcribe an asset with a different model or language so I can improve the transcript quality.

**US-232**  
As a user, I want to be informed when no model is installed so I know what to do before transcription is available.
